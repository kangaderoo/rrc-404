A site to perform the Freeze and Melt action on the Water(ICE) token

Ice/Water tokens are an experimental version of ERC-404 built on Radix with Scrypto.
Token holders can convert between Ice NFTs and Water Fungibles.

When Minting the NFT a green, blue, red, orange or purple result could be obtained.
There is no randomness in this assignment.
The components distribution algoritmn aims to get the following prefered distribution:
Green should be 42% of the circulating NFTs, Blue 30%, Red 16%, Orange 8% and Purple 4%.
If the precentile amount of any color is less then the target percentile the color with the
highest percentile deficit will be mined next.
Both Freezing and Melting affects the ratio's of the current distribution.
The actual minted percentages are listed on this page.
